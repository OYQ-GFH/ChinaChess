#### 项目简介

> python实现中国象棋 基于pygame模块开发

#### 项目结构

```python
/ images  # 图片素材
/ musics  # 音乐素材
/ XQlightPy # AI引擎模块

chinese_chess.py  # 主程序
play_against_ai.py # AI对战模式
```
